# Spotifake
my take on creating a Spotify-look-alike.

## Features
```
core:
[v] right 'more' menu
[v] album covers slider
[v] play button with state
[v] playlist menu
[v] responsiveness

additional:
[v] btn
[v] deal with the computed watcher console.error
[v] flow control
[v] toggleable buttons
[v] dataset from outside
```

## Project setup
```
yarn / npm i
```

### Run and see
```
yarn serve / npm run serve
```
